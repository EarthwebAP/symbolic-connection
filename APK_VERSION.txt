================================================================================
INSTALLATION & USAGE
================================================================================

1. Download APK:
   wget https://your-domain/downloads/symbolic-connection.apk

2. Install on Android Device:
   adb install symbolic-connection.apk

3. Launch Application:
   App will automatically register with telemetry server on first launch
   Device ID extracted from Android secure settings (ANDROID_ID)
   Heartbeats sent every 5 seconds while app is running

4. Monitor Telemetry:
   View real-time metrics at: http://localhost:9998/api/apk/metrics
   Dashboard: http://your-domain:3033/apk-dashboard.html

================================================================================
SECURITY & PRIVACY
================================================================================

Data Collection:
  - Device ID (from ANDROID_ID)
  - Device model, brand, Android version
  - App version and permissions
  - Network connection type
  - Geographic location (IP-based)
  - Event timestamps and metadata

Privacy Measures:
  - Device ID is anonymized via SHA256 hashing in backend
  - No PII (Personally Identifiable Information) collected
  - No message content or media stored
  - No user credentials retained
  - Geographic data limited to city-level precision
  - Users can opt-out by disabling in settings

KNOWN LIMITATIONS
================================================================================

Telemetry Endpoint:
  - Currently configured to localhost:9998
  - Must be updated for production deployments
  - Requires network connectivity for event transmission

Geolocation:
  - IP-based geolocation may be inaccurate
  - Device must have internet connectivity
  - Subject to ISP and VPN masking

Heartbeat Interval:
  - Fixed at 5-second intervals while app active
  - May increase battery usage on low-end devices
  - Can be adjusted in SymbolicConnectionClient.kt
